# EDA와 시각화

## EDA

Exploratory Data Analysis : 데이터를 수집하고 정제한 후 이루어지는 데이터 분석의 매우 중요한 초기 단계

[이점]

1. 데이터의 잠재적인 문제 발견
2. 데이터의 다양한 패턴 발견
3. 자료수집을 위한 기반
4. 적절한 통계 도구 제시

[대상]

- 일변량(Univariate)

  독립변수 **1개** $\rightarrow$ 종속변수 **1개**

  > 데이터를 설명하고 패턴을 찾는 것이 목적

- 다변량(Mulit-variate)

  독립변수 **여러개** $\rightarrow$ 종속변수 **여러개**

  > 변수들 간의 관계를 보는 것이 목적
